# Remove background

Este proyecto usa la librería `rembg` de python para hacer la tarea principal de remover el background de la foto que tu desees.

# Requirements

``` python
python: >3.7, <3.13
```

Librerías usadas:
 - os
 - numpy
 - cv2
 - skimage
 - tkinter
 - rembg
